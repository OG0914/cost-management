# 系统配置

## 功能概述

系统配置用于管理业务参数和基础数据配置，包括费率设置、运费配置、利润档位和原料类别管理。

---

## 页面布局

页面分为两个 Tab：
- **业务配置**：费率、运费、利润等业务参数
- **基础数据配置**：原料类别管理

---

## 业务配置

### 基础费率

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 管销率 | 管理和销售费用占比 | 20% |
| 增值税率（默认） | 默认增值税率 | 13% |
| 保险率 | 保险费用占比 | 0.3% |

**增值税率选项**：
- 可配置多个增值税率选项供选择
- 支持添加、删除选项
- 至少保留一个选项

### 汇率与系数

| 参数 | 说明 |
|------|------|
| 汇率 CNY/USD | 人民币兑美元汇率 |
| 工价系数 | 工序总价计算系数（工序总价 = 小计 × 系数） |
| FOB深圳汇率 | FOB深圳专用汇率 |

### 整柜运费 FCL

| 参数 | 说明 | 单位 |
|------|------|------|
| 20尺整柜 | 20尺集装箱运费 | USD |
| 40尺整柜 | 40尺集装箱运费 | USD |

### 散货运费 LCL

**基础运费按体积区间**：

| 体积区间 | 运费 |
|----------|------|
| 1-3 CBM | 基础运费 |
| 3-5 CBM | 基础运费 |
| 5-8 CBM | 基础运费 |
| 8-10 CBM | 基础运费 |
| 10-12 CBM | 基础运费 |
| 12-15 CBM | 基础运费 |

**其他费用**：
- 操作费 Handling
- 拼箱费 CFS/CBM
- 文件费

### CIF 深圳运费设置

| 参数 | 说明 | 单位 |
|------|------|------|
| 海运费 Sea Freight | 海运费用 | USD/CBM |
| CFS 费用 | 集装箱货运站费用 | CNY/CBM |
| 文件费 DOC FEE | 单证费用 | CNY |
| 报关费 Customs | 报关代理费用 | CNY |

| 湖北知腾卡车费 | 湖北工厂卡车运费 | CNY/CBM |

### 利润档位

- 可配置多个利润率档位
- 用于成本分析时选择目标利润率
- 支持添加、删除档位
- 至少保留一个档位

## 基础数据配置

### 原料类别管理

用于设置原料的分类，在原料管理中用于筛选和分组。

**操作步骤**：

1. 切换到 **【基础数据配置】** Tab
2. 在原料类别管理区域：
   - 查看现有类别列表
   - 编辑类别名称和排序
   - 删除不需要的类别
3. 点击 **【新增类别】** 按钮添加新类别
4. 填写类别名称和排序号
5. 点击 **【保存类别配置】** 保存修改

---

## 操作指南

### 保存配置

1. 修改需要的参数值
2. 点击页面底部 **【保存业务配置】** 按钮
3. 系统校验后保存，显示成功提示

### 离开页面提醒

- 如有未保存的修改，离开页面时会提示确认
- 浏览器关闭或刷新前也会提醒

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 配置生效时间 | 配置保存后立即生效，只影响新创建的成本分析 |
| 费率范围 | 费率类参数必须在 0-1 之间（0%-100%） |
| 汇率必须大于0 | 汇率类参数必须大于 0 |
| 至少一个选项 | 增值税率选项和利润档位至少保留一个 |
| 类别名称唯一 | 原料类别名称不能重复 |
