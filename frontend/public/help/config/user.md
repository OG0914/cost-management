# 用户管理

## 功能概述

用户管理用于管理系统用户账号，包括新增、编辑、删除用户，以及重置密码等操作。

---

## 页面布局

页面分为筛选区、用户展示区和操作区：
- **筛选区**：搜索框（支持用户代号、真实姓名搜索）、视图切换按钮
- **用户展示区**：卡片视图或列表视图展示用户数据
- **操作区**：右上角工具栏（展开后显示全部功能按钮）

---

## 用户信息字段

| 字段名 | 说明 |
|--------|------|
| 用户代号 | 登录系统的用户名 |
| 真实姓名 | 用户的真实姓名 |
| 角色 | 用户所属角色（管理员/采购/生产/审核/业务/只读） |
| 邮箱 | 用户邮箱地址 |
| 状态 | 已启用/已禁用 |

---

## 角色说明

| 角色 | 颜色标识 | 说明 |
|------|----------|------|
| 管理员 | 红色 | 系统最高权限 |
| 采购 | 橙色 | 负责原料采购管理 |
| 生产 | 绿色 | 负责生产管理 |
| 审核 | 蓝色 | 负责审核成本分析 |
| 业务 | 紫色 | 负责业务报价 |
| 只读 | 灰色 | 仅查看权限 |

## 操作指南

### 1. 新增用户

1. 点击页面右上角 **【新增用户】** 按钮
2. 填写用户信息：
   - **用户代号**（必填）：3-20个字符，用于登录系统
   - **密码**（必填）：至少6个字符
   - **真实姓名**（必填）：用户的真实姓名
   - **角色**（必填）：选择用户角色
   - **邮箱**：可选，用于接收通知
3. 点击 **【确定】** 完成创建

### 2. 编辑用户

1. 在用户列表中找到目标用户
2. 点击操作列的 **【编辑】** 按钮
3. 修改需要更新的字段：
   - 用户代号
   - 真实姓名
   - 角色
   - 邮箱
   - 状态（启用/禁用）
4. 点击 **【确定】** 确认修改

### 3. 重置密码

1. 找到需要重置密码的用户
2. 点击操作列的 **【重置密码】** 按钮（钥匙图标）
3. 输入新密码（至少6个字符）
4. 点击 **【确定】** 完成重置

### 4. 删除用户

1. 找到需要删除的用户
2. 点击操作列的 **【删除】** 按钮
3. 确认删除操作

⚠️ **注意**：用户代号为 "admin" 的管理员账号不可删除。

### 5. 导出Excel

1. 在列表视图中勾选需要导出的用户（使用左侧复选框）
2. 点击工具栏的 **【导出Excel】** 按钮
3. 系统自动下载选中数据为Excel文件

⚠️ **注意**：需先勾选数据才能导出。

### 6. 导入Excel

1. 点击工具栏的 **【下载模板】** 获取导入模板
2. 按模板格式填写用户数据
3. 点击 **【导入Excel】** 按钮，选择填写好的文件
4. 系统校验后完成导入，显示导入结果（创建条数、更新条数、错误信息）

## 视图模式

### 卡片视图
- 以卡片形式展示用户
- 显示头像（真实姓名首字母）、用户代号、真实姓名、角色标签、邮箱、状态
- 操作按钮：重置密码、编辑、删除

### 列表视图
- 以表格形式展示用户
- 支持多选（用于批量导出）
- 显示所有字段信息

---

## 搜索功能

### 搜索框
支持按用户代号、真实姓名进行模糊搜索。

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| admin账号保护 | 系统默认管理员账号不可删除，确保系统始终有管理员可用 |
| 用户代号唯一 | 用户代号不能重复，创建时会校验唯一性 |
| 密码安全 | 建议设置复杂度较高的密码，定期更换 |
| 离职处理 | 员工离职时建议禁用账号或删除账号 |
| 角色选择 | 请根据实际工作职责分配合适的角色 |
