# 权限管理

## 功能概述

权限管理用于配置系统用户和角色权限。

## 角色说明

| 角色 | 权限范围 |
|------|----------|
| 管理员 | 所有功能 |
| 审核员 | 成本分析、审核管理、基础数据 |
| 业务员 | 成本分析、审核管理、基础数据 |
| 采购/生产 | 基础数据管理 |
| 只读用户 | 查看成本分析、基础数据 |

## 功能说明

### 用户管理
- 新增用户
- 编辑用户
- 重置密码
- 启用/禁用账号

### 角色管理
- 查看角色权限
- 分配用户角色

## 注意事项

- 请妥善保管管理员账号
- 离职员工请及时禁用账号
