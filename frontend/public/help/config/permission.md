# 权限管理

## 功能概述

权限管理用于为不同角色配置系统的功能访问权限。管理员可以为每个角色精细控制可访问的模块和操作。

---

## 页面布局

页面分为左右两部分：
- **左侧**：角色列表，显示系统中所有角色
- **右侧**：权限配置面板，为选中的角色配置权限

---

## 角色说明

| 角色 | 代码 | 说明 |
|------|------|------|
| 管理员 | admin | 系统最高权限，拥有所有功能的访问和操作权限 |
| 采购 | purchaser | 负责原料采购管理，可管理原料数据，查看基础数据 |
| 生产 | producer | 负责生产管理，可管理工序和包材，查看基础数据 |
| 审核 | reviewer | 负责审核成本分析，可批准或退回报价，查看成本数据 |
| 业务 | salesperson | 负责业务报价，可创建和管理成本分析，管理客户 |
| 只读 | readonly | 仅查看权限，无法修改任何数据，适合访客使用 |

---

## 操作指南

### 选择角色

1. 在左侧角色列表中点击需要配置的角色
2. 右侧显示该角色的权限配置面板
3. 管理员角色显示特殊提示，不可修改权限

### 配置权限

**单个权限切换**：
1. 在权限列表中找到需要配置的权限项
2. 勾选或取消勾选该权限
3. 权限变更会实时显示在界面上

**批量操作**：

| 操作 | 说明 |
|------|------|
| 展开全部 | 展开所有权限分组 |
| 折叠全部 | 折叠所有权限分组 |
| 全选所有权限 | 为当前角色授予所有权限 |
| 清空所有 | 移除当前角色的所有权限 |

**分组操作**：
- 点击分组标题栏的全选框，可快速选中/取消该分组下的所有权限
- 分组显示部分选中状态时，全选框显示为半选状态

### 保存权限

1. 完成权限调整后，点击右上角的 **【保存】** 按钮
2. 系统保存该角色的权限配置
3. 显示保存成功提示

### 重置权限

1. 点击右上角的 **【重置】** 按钮
2. 确认重置操作
3. 权限恢复为上次保存的状态

---

## 权限分组

权限按功能模块分组，常见分组包括：

- **成本分析**：创建、编辑、查看、删除成本分析
- **审核管理**：审批、驳回、查看审核列表
- **基础数据**：客户、物料、型号、包材、工序、法规管理
- **系统配置**：系统参数、权限管理、用户管理

---

## 权限生效说明

| 说明项 | 详情 |
|--------|------|
| 生效时间 | 权限修改保存后立即生效 |
| 已登录用户 | 已登录用户需要重新登录才能获取最新权限 |
| 管理员 | 管理员角色拥有所有权限，不可修改 |

---

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 谨慎授权 | 请根据实际工作需要分配权限，避免过度授权 |
| 权限测试 | 修改权限后建议用该角色账号测试验证 |
| 管理员保护 | 管理员权限不可修改，确保系统始终有管理员可用 |
| 离职处理 | 员工离职时建议禁用账号或移除敏感权限 |
