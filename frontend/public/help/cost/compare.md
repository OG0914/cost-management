# 成本对比

## 功能概述

成本对比功能允许同时比较多个成本分析（2-4个），并排展示各方案的成本构成差异，便于分析不同配置、数量、客户对成本的影响。

---

## 操作步骤

### 1. 选择对比记录

1. 进入【成本记录】页面
2. 在列表中勾选需要对比的记录（至少2个，最多4个）
3. 只能选择状态为"已提交"或"已审核"的记录
4. 点击页面顶部的【对比模式】按钮
5. 进入成本对比分析页面

### 2. 查看对比结果

对比页面分为两个区域：

#### 对比概览（上方）

并排显示各记录的核心信息：
- 成本分析编号
- 状态标签
- 销售类型
- 客户名称
- 产品型号
- 包装方式
- 订单数量
- 最终成本价

#### 详细对比表格（下方）

按类别对比各项成本：

| 对比项 | 说明 |
|--------|------|
| 基本信息 | 型号、客户、数量、销售类型等 |
| 原料成本 | 各原料明细及小计对比 |
| 工序成本 | 各工序明细及小计对比 |
| 包材成本 | 包材配置及成本对比 |
| 运费 | 运输方式和费用对比 |
| 费用明细 | 管销费、增值税/保险等 |
| 利润分析 | 各利润率档位对比 |

---

## 导出与打印

### 导出Excel

点击【导出Excel】按钮，将对比结果导出为Excel文件。

导出内容包括：
- 各记录的基本信息
- 成本明细对比
- 利润分析对比

### 打印

点击【打印】按钮，可将对比结果打印或保存为PDF。

打印样式已优化：
- 隐藏操作按钮
- 避免表格分页截断
- 适合A4纸张打印

## 对比分析建议

### 有效对比场景

1. **相同型号不同数量**
   - 分析规模效应对单价的影响
   - 找出最优经济订货量

2. **相同型号不同客户**
   - 分析客户特定要求（包装、法规）对成本的影响
   - 制定差异化报价策略

3. **相同型号不同时期**
   - 追踪原料价格波动对成本的影响
   - 分析成本变动趋势

4. **相同型号不同运输方式**
   - 对比LCL/FCL/CIF等不同运输方式的成本差异
   - 选择最优物流方案

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 数量限制 | 最少2个，最多4个记录同时对比 |
| 状态限制 | 只能选择"已提交"或"已审核"的记录 |
| 币种差异 | 注意内销（CNY）和外销（USD）的币种差异 |
| 型号差异 | 不同型号的对比意义有限，建议优先对比相同型号 |

## 常见问题

**Q: 为什么"对比模式"按钮是灰色的？**
A: 需要至少选择2个记录才能启用对比功能。

**Q: 最多可以对比几个记录？**
A: 最多可同时对比4个成本分析记录。

**Q: 为什么有些记录无法勾选？**
A: 只有"已提交"或"已审核"状态的记录可以用于对比，草稿和已退回的记录不可选。

**Q: 可以对比不同型号的记录吗？**
A: 可以，但对比结果的意义可能不大，建议优先对比相同型号以分析其他变量（数量、客户、运输方式）的影响。
