# 成本记录

## 功能概述

成本记录页面展示所有已创建的成本分析，支持搜索、筛选和对比，方便管理和查找历史记录。

---

## 页面布局

- **顶部工具栏**：搜索框、筛选条件、功能按钮
- **数据展示区**：表格形式展示成本分析记录
- **分页控制**：底部 pagination 控件

---

## 搜索与筛选

### 关键词搜索

在搜索框输入关键词，系统搜索以下字段：
- 成本分析编号
- 客户名称
- 产品型号

搜索支持防抖（300ms），自动触发查询。

### 状态筛选

| 状态 | 说明 |
|------|------|
| 草稿 | 正在编辑，未提交 |
| 已提交 | 已提交，等待审核 |
| 已审核 | 审核已通过 |
| 已退回 | 审核未通过，可修改 |

### 销售类型筛选

- 内销：国内销售（CNY）
- 外销：出口销售（USD）

---

## 列表字段

| 字段 | 说明 |
|------|------|
| 成本分析编号 | 系统生成的唯一编号 |
| 状态 | 当前审核状态（带标签显示） |
| 销售类型 | 内销/外销 |
| 客户名称 | 关联的客户 |
| 客户地区 | 客户所在地区 |
| 产品类别 | 口罩/半面罩/全面罩/PPE产品 |
| 产品型号 | 具体型号名称 |
| 包装方式 | 包材配置名称及规格 |
| 订单数量 | 产品数量（件/箱） |
| 最终成本价 | 计算后的最终价格（含币种） |
| 创建人 | 创建该记录的用户 |
| 创建时间 | 记录创建时间 |

**特殊标记**：
- **预估**：新产品成本预估模式创建的记录
- **标准**：已设为标准成本的记录

---

## 记录操作

| 操作 | 适用状态 | 说明 |
|------|----------|------|
| 查看 | 全部 | 查看成本分析详情 |
| 编辑 | 草稿/已退回 | 修改成本分析内容 |
| 复制 | 全部 | 复制创建新的成本分析 |
| 删除 | 草稿（管理员可删全部） | 删除该记录 |

**权限控制**：
- 管理员/审核员/只读角色：不显示【新增成本分析】按钮
- 受限角色：部分操作按钮隐藏

---

## 对比模式

### 选择记录

1. 在列表左侧勾选需要对比的记录
2. 只可选择状态为"已提交"或"已审核"的记录
3. 至少选择2条记录
4. 最多选择4条记录

### 进入对比

1. 选择记录后，【对比模式】按钮变为可用
2. 按钮显示已选择数量，如"对比模式 (2)"
3. 点击按钮进入成本对比分析页面

---

## 新增成本分析

1. 点击页面右上角【新增成本分析】按钮
2. 弹出产品类别选择弹窗
3. 选择产品类别（口罩/半面罩/全面罩/PPE产品）
4. 进入成本分析创建页面

---

## 移动端适配

在移动设备上，表格切换为卡片列表展示：
- 显示编号、状态、客户、型号、价格等关键信息
- 支持点击查看详情
- 操作按钮简化显示
