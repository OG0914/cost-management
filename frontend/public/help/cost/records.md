# 成本记录

## 功能概述

成本记录页面展示所有已创建的成本分析，支持查询、筛选和导出。

## 功能说明

### 查询筛选
- 按型号查询
- 按状态筛选（草稿/待审核/已通过/已退回）
- 按日期范围筛选
- 按创建人筛选

### 列表操作
- 查看详情
- 编辑（草稿/已退回状态）
- 删除（草稿状态）
- 导出Excel

### 状态说明

| 状态 | 说明 |
|------|------|
| 草稿 | 正在编辑，未提交 |
| 待审核 | 已提交，等待审核 |
| 已通过 | 审核通过 |
| 已退回 | 审核未通过，可修改后重新提交 |
