# 编辑成本分析

## 功能概述

编辑成本分析允许修改已创建但未提交审核的成本分析，或对已退回的成本分析进行修改后重新提交。

---

## 进入编辑页面

1. 点击左侧菜单 **【成本记录】**
2. 在列表中找到需要编辑的记录
3. 确认记录状态为"草稿"或"已退回"
4. 点击操作列的 **【编辑】** 按钮
5. 进入成本分析编辑页面

---

## 退回修改流程

当成本分析被审核退回时：

1. **查看退回原因**
   - 页面顶部显示黄色警告提示
   - 提示内容包含审核退回原因

2. **针对性修改**
   - 根据退回原因定位问题项
   - 修改相应内容
   - 可随时点击"保存草稿"

3. **重新提交**
   - 确认修改完成后，点击"提交审核"
   - 状态变为"已提交"，进入审核流程

---

## 可编辑内容

### 草稿状态

| 类别 | 可编辑项 | 说明 |
|------|----------|------|
| 基本信息 | 客户名称、客户地区、销售类型 | 法规标准和型号配置可修改（如未绑定配置） |
| 原料配置 | 原料选择、单价、用量、管销后标记 | 可增删改，支持锁定/解锁编辑 |
| 工序配置 | 工序名称、单价、数量 | 可增删改，支持锁定/解锁编辑 |
| 包材配置 | 包材选择、单价、用量 | 可增删改，支持锁定/解锁编辑 |
| 费用参数 | 运费、管销后费用项 | 可修改 |
| 利润档位 | 自定义利润档位 | 可增删改 |

### 已退回状态

与草稿状态基本相同，但：
- **法规标准**和**型号配置**字段被锁定（不可修改）
- 如需更换型号，需要重新创建成本分析

---

## 明细编辑模式

### 锁定/解锁功能

成本明细（原料/工序/包材）支持锁定/解锁编辑：

- **锁定状态**：名称字段只读，防止误修改
- **解锁状态**：可编辑名称字段，用于自定义项

**操作**：点击明细表格上方的编辑按钮切换锁定状态。

---

## 保存与提交

### 保存草稿
- 点击 **【保存草稿】** 保存修改但不提交
- 系统每30秒自动保存
- 离开页面时提示保存

### 提交审核
- 点击 **【提交审核】** 保存并提交审核
- 提交后不可编辑，需等待审核结果

### 取消
- 点击 **【取消】** 返回成本记录页面

---

## 注意事项

| 可修改 | 不可修改 |
|--------|----------|
| ✓ 客户、销售类型、数量 | ✗ 已提交或已审核的状态 |
| ✓ 原料、工序、包材配置 | ✗ 已退回记录的型号配置 |
| ✓ 费用和利润档位 | ✗ 其他用户的记录（除非管理员） |

---

## 常见问题

**Q: 为什么有些记录没有编辑按钮？**
A: 只有状态为"草稿"或"已退回"且当前用户为创建者或管理员时，才显示编辑按钮。

**Q: 审核通过后发现有错误怎么办？**
A: 已审核通过的成本分析无法直接修改，需要：
1. 联系管理员将该记录退回
2. 或复制该记录创建新的成本分析

**Q: 修改后忘记保存怎么办？**
A: 系统每30秒自动保存草稿，离开页面时也会提示保存。
