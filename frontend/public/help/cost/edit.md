# 编辑成本分析

## 功能概述

编辑成本分析功能允许您修改已创建但未提交审核的成本分析。

## 操作步骤

1. 进入"成本记录"页面
2. 找到需要编辑的记录（状态为"草稿"或"已退回"）
3. 点击"编辑"按钮
4. 修改需要调整的内容
5. 点击"保存"或"提交审核"

## 可编辑内容

- 基本信息（型号、数量、销售类型等）
- 原料配置及用量
- 包材配置
- 工序设置
- 费用参数

## 注意事项

- 已提交审核的记录无法直接编辑
- 已审核通过的记录需要重新创建
- 编辑后记得保存更改
