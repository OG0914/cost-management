# 创建成本分析

## 功能概述

创建成本分析用于计算产品的完整成本构成，生成报价单。支持两种模式：
- **标准模式**：选择已有型号和包装配置
- **新产品预估模式**：针对尚未建立完整BOM的新产品进行成本预估

---

## 进入创建页面

### 方式一：从成本记录页面进入
1. 点击左侧菜单 **【成本记录】**
2. 点击页面右上角 **【新增成本分析】** 按钮
3. 选择产品类别（口罩/半面罩/全面罩/PPE产品）
4. 进入创建页面

### 方式二：新产品预估模式
1. 在新增成本分析时，部分产品类别支持选择 **【新产品预估】**
2. 进入预估模式，可选择新产品型号并参考标准成本

---

## 页面布局

页面采用左右分栏布局：
- **左侧**：表单填写区（基本信息、销售类型、成本明细）
- **右侧**：成本预览面板（实时显示计算结果）

---

## 填写步骤

### 1. 基本信息

| 字段 | 必填 | 说明 |
|------|------|------|
| 法规标准 | 是 | 选择适用的法规（GB/NIOSH/CE等） |
| 型号配置 | 是 | 选择产品型号和包装配置（标准模式） |
| 新产品型号 | 是 | 选择新产品型号（预估模式） |
| 参考标准成本 | 否 | 选择同法规同分类的标准成本作为参考（预估模式） |
| 客户名称 | 否 | 输入或选择客户，支持搜索现有客户 |
| 客户地区 | 否 | 自动填充或手动输入 |

**客户搜索功能**：
- 输入客户名称时自动搜索匹配的客户
- 显示客户VC号、地区、负责业务员
- 新客户：标记为"新客户"
- 他人客户：标记为"他人客户"（使用其他业务员的客户）

### 2. 销售类型

选择销售类型卡片：

| 类型 | 币种 | 税率 |
|------|------|------|
| **内销** | CNY | 含增值税（可配置税率选项） |
| **外销** | USD | FOB条款，0%税率 |

**内销特有配置**：
- 增值税率选择：从系统配置的税率选项中选择
- 数量输入：支持按件或按箱输入
- 运费配置：按CBM计价

**外销特有配置**：
- 运输方式：
  - FCL（整柜）：20尺/40尺
  - LCL（散货）：按CBM计费
- 港口类型：
  - FOB深圳
  - CIF深圳
- 数量输入：支持按件/箱/托盘输入
- 目的港：输入目的港口名称

### 3. 成本明细

成本明细分为三个Tab：原料、工序、包材

#### 原料配置

- 从原料库选择原料（支持搜索）
- 自动填充单价
- 输入用量
- 支持设置"管销后计算"（该原料在管销费后计入）
- 支持锁定/解锁编辑模式

**操作**：
- 添加行：点击"添加原料"
- 删除行：点击行尾删除按钮
- 搜索原料：输入关键词搜索原料库

#### 工序配置

- 添加工序项
- 输入工序名称、单价、数量
- 系统自动计算小计（单价 × 数量 × 工价系数）
- 支持锁定/解锁编辑模式

#### 包材配置

- 从包材库选择包材
- 自动填充单价
- 输入用量和外箱体积
- 支持锁定/解锁编辑模式

---

## 成本预览面板（右侧）

右侧面板实时显示成本计算结果：

### 基础成本
- 原料小计
- 工序小计
- 包材小计

### 费用计算
- 基础成本合计
- 运费（根据销售类型和运输方式计算）
- 管销价（基础成本 + 运费 + 管销费）
- 最终成本价
  - 内销：含增值税价格
  - 外销：含保险价格（USD）

### 利润分析
- 系统利润档位：显示各利润率对应的建议售价
- 自定义利润档位：可添加自定义利润率
- 利润计算器：实时计算不同利润率下的售价和利润额

### 自定义费用
- 可添加管销后费用项（如：佣金、折扣等）
- 设置费率，自动计算费用金额

---

## 保存与提交

### 保存草稿
- 点击 **【保存草稿】** 按钮
- 数据保存为草稿状态，可随时修改
- 系统每30秒自动保存草稿
- 离开页面时提示保存草稿

### 提交审核
- 点击 **【提交审核】** 按钮
- 状态变为"已提交"，进入审核流程
- 提交后不可编辑，需等待审核结果

### 取消
- 点击 **【取消】** 按钮
- 提示确认后清除当前填写内容

---

## 草稿自动保存

- 系统每30秒自动保存草稿
- 重新进入创建页面时，提示恢复草稿
- 可手动选择放弃或恢复草稿

---

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 型号锁定 | 编辑模式下，法规标准和型号配置不可修改 |
| 客户地区 | 选择已有客户时自动填充地区 |
| 运费计算 | 根据销售类型和运输方式自动计算 |
| 数量单位 | 注意切换数量单位（件/箱/托盘） |
| 工价系数 | 工序总价 = 小计 × 系统配置的工价系数 |
