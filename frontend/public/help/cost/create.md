# 创建成本分析

本文介绍如何创建一个新的成本分析。

## 操作步骤

### 1. 进入创建页面

点击左侧菜单"新增成本分析"，进入成本分析创建页面。

### 2. 填写基本信息

| 字段 | 说明 | 必填 |
|------|------|------|
| 型号 | 选择要分析的产品型号 | 是 |
| 数量 | 输入产品数量 | 是 |
| 销售类型 | 选择国内销售或出口 | 是 |
| 客户 | 选择客户（可选） | 否 |
| 法规 | 选择适用的法规标准 | 是 |

### 3. 配置原料

1. 在原料配置区域点击"添加原料"
2. 从原料库中选择需要的原料
3. 输入每种原料的用量
4. 系统会自动计算原料成本

### 4. 配置包材

1. 选择包材类型和规格
2. 系统会根据型号自动推荐包材配置
3. 可以手动调整包材数量

### 5. 添加工序

1. 点击"添加工序"
2. 选择需要的生产工序
3. 设置工序数量和单价

### 6. 设置费用

根据销售类型配置相应费用：

- **国内销售**：配置国内运费
- **出口**：配置国际运费（CIF/LCL/FCL）

### 7. 计算成本

点击"计算成本"按钮，系统会生成完整的成本分析报告。

## 注意事项

- 创建前请确保基础数据（原料、包材、工序）已维护
- 法规选择会影响可用的原料和检测要求
- 建议在提交前仔细检查各项配置

## 常见问题

**Q: 为什么无法选择某些原料？**
A: 请检查所选法规是否允许使用该原料，以及原料状态是否为"启用"。

**Q: 如何保存为模板？**
A: 成本计算完成后，可以点击"保存为标准成本"将当前配置保存为模板。
