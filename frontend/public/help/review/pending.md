# 待审核记录

## 功能概述

待审核页面展示所有已提交但尚未审核的成本分析记录。审核人员可以查看记录详情并执行审核操作（通过或退回），业务员可以查看自己提交的待审核记录。

---

## 页面布局

- **搜索区**：关键词搜索框
- **数据列表**：表格形式展示待审核记录
- **分页控制**：底部分页器

---

## 搜索功能

在搜索框输入关键词，系统搜索以下字段：
- 成本分析编号
- 客户名称
- 产品型号

搜索支持防抖（300ms），自动触发查询。清空搜索框时立即刷新列表。

---

## 列表字段

| 字段 | 说明 |
|------|------|
| 成本分析编号 | 系统生成的唯一编号 |
| 状态 | 当前为"已提交" |
| 销售类型 | 内销/外销 |
| 客户名称 | 关联的客户 |
| 产品型号 | 具体型号名称 |
| 包装方式 | 包材配置名称及规格 |
| 订单数量 | 产品数量（件/箱） |
| 最终成本价 | 计算后的最终价格（含币种） |
| 创建人 | 提交审核的用户 |
| 提交时间 | 提交审核的日期时间 |

---

## 操作按钮

根据用户角色显示不同操作：

| 角色 | 操作按钮 | 说明 |
|------|----------|------|
| 管理员/审核员 | 审核 | 打开审核弹窗，可执行通过/退回操作 |
| 业务员 | 查看 | 打开查看弹窗，仅可查看不可操作 |
| 管理员 | 删除 | 删除该记录（需谨慎） |

---

## 审核操作

### 打开审核弹窗

1. 在列表中找到目标记录
2. 点击"审核"按钮（管理员/审核员）
3. 打开审核详情弹窗

### 审核弹窗内容

- **左侧**：成本分析完整信息
  - 基本信息（型号、客户、数量、销售类型等）
  - 原料明细
  - 工序明细
  - 包材明细
  - 费用计算
- **右侧**：审核操作区
  - 批注输入框
  - "通过"按钮
  - "退回"按钮

### 审核通过

1. 核对成本分析内容无误
2. 可选择性填写批注（如"审核通过"、"价格已确认"等）
3. 点击"通过"按钮
4. 记录状态变为"已通过"
5. 列表自动刷新

### 审核退回

1. 发现问题需要修改
2. **必须填写退回原因**（批注框）
3. 点击"退回"按钮
4. 记录状态变为"已退回"
5. 创建人可在"成本记录"页面编辑修改

**常用退回原因**：
- 原料单价异常，请核实
- 工序配置不完整，请补充
- 客户信息缺失，请补充
- 数量填写有误，请检查
- 法规选择不正确，请修改
- 运费计算有误，请重新核算

---

## 审核要点

审核时请重点关注：

- **基本信息完整**：型号、客户、数量是否填写正确
- **原料配置合理**：原料选择是否正确，单价是否符合市场行情
- **包材配置正确**：包材类型和规格是否匹配型号
- **工序设置完整**：工序流程是否完整，价格是否合理
- **费用计算准确**：运费、管销费等是否计算正确
- **法规符合性**：所选法规是否适用于目标市场
- **利润率合理**：建议售价是否符合公司定价策略

---

## 移动端适配

在移动设备上：
- 表格切换为卡片列表展示
- 显示核心信息：编号、状态、客户、型号、价格
- 点击卡片可直接进入审核/查看流程
