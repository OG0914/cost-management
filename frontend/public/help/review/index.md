# 审核管理

审核管理模块用于处理成本分析的审核流程，确保报价单的准确性和合规性。

## 功能概述

- 查看待审核的成本分析
- 执行审核操作（通过/退回）
- 查看已审核记录（已通过/已退回）

## 审核流程

```
提交审核 → 待审核 → 审核通过/退回
                    ↓
              已通过/已退回
```

## 角色权限

- **审核员**：可以执行审核操作（通过/退回）
- **业务员**：可以查看待审核和已审核记录（只读）
- **管理员**：拥有所有权限
