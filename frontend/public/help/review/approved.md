# 已审核

## 功能概述

已审核页面展示所有已完成审核的成本分析记录，包括已通过和已退回的。

## 功能说明

### 列表展示
- 显示所有已审核记录
- 支持按状态筛选（已通过/已退回）
- 支持按审核日期筛选

### 查看详情
- 点击记录查看完整成本分析
- 已退回记录可查看退回原因

### 导出功能
- 支持导出Excel报表
- 支持批量导出

## 状态说明

| 状态 | 说明 |
|------|------|
| 已通过 | 审核通过，可作为正式报价单 |
| 已退回 | 审核未通过，需修改后重新提交 |
