# 已审核记录

## 功能概述

已审核页面展示所有已完成审核的成本分析记录，包括已通过和已退回两种状态。所有角色均可查看，管理员可删除记录。

---

## 页面布局

- **搜索区**：关键词搜索框
- **数据列表**：表格形式展示已审核记录
- **分页控制**：底部分页器

---

## 搜索功能

在搜索框输入关键词，系统搜索以下字段：
- 成本分析编号
- 客户名称
- 产品型号

搜索支持防抖（300ms），自动触发查询。清空搜索框时立即刷新列表。

---

## 列表字段

| 字段 | 说明 |
|------|------|
| 成本分析编号 | 系统生成的唯一编号 |
| 状态 | 已通过/已退回 |
| 批注 | 审核时填写的意见或退回原因（截取前10字） |
| 销售类型 | 内销/外销 |
| 客户名称 | 关联的客户 |
| 产品型号 | 具体型号名称 |
| 包装方式 | 包材配置名称及规格 |
| 订单数量 | 产品数量（件/箱） |
| 最终成本价 | 计算后的最终价格（含币种） |
| 审核人 | 执行审核的用户 |
| 审核时间 | 审核完成的日期时间 |

### 批注显示说明

- 列表中显示批注前10个字符，超过显示省略号
- 鼠标悬停可查看完整批注内容（Tooltip显示）
- 退回原因的【退回原因】前缀会自动移除，只显示原因内容

---

## 状态说明

| 状态 | 说明 | 后续操作 |
|------|------|----------|
| 已通过 | 审核通过，可作为正式报价单使用 | 可在成本记录中查看、复制 |
| 已退回 | 审核未通过，需修改后重新提交 | 可在成本记录中编辑修改 |

---

## 操作按钮

| 角色 | 操作按钮 | 说明 |
|------|----------|------|
| 全部角色 | 查看 | 打开详情弹窗查看完整信息 |
| 管理员 | 删除 | 删除该记录（需谨慎） |

---

## 查看详情

1. 在列表中找到目标记录
2. 点击"查看"按钮
3. 打开已审核详情弹窗，查看完整的成本分析信息

**详情弹窗内容**：
- 基本信息（编号、状态、型号、客户等）
- 审核信息（审核人、审核时间、批注）
- 成本明细（原料、工序、包材）
- 费用计算
- 利润分析

---

## 已退回记录的后续处理

### 查看退回原因

1. 在已审核列表中找到状态为"已退回"的记录
2. 查看批注列的退回原因（悬停可看完整内容）
3. 或点击"查看"按钮，在详情弹窗中查看完整批注

### 修改成本分析

1. 进入"成本记录"页面
2. 找到该退回记录（状态为"已退回"）
3. 点击"编辑"按钮
4. 根据退回原因修改相应内容
5. 修改完成后点击"提交审核"重新提交

### 注意事项

- 已退回记录可以直接编辑，无需复制创建新记录
- 编辑时**型号配置字段会被锁定**，无法修改
- 如需更换型号，需要重新创建成本分析
- 重新提交后，原退回记录保留在"已退回"列表中作为历史记录

---

## 移动端适配

在移动设备上：
- 表格切换为卡片列表展示
- 显示核心信息：编号、状态、客户、型号、价格
- 点击卡片可查看详情
