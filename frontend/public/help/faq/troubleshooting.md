# 故障排除

## 🔍 快速排查清单

遇到问题时，建议按以下顺序排查：

- [ ] **刷新页面**（F5 或 Ctrl+R）
- [ ] **清除浏览器缓存**（Ctrl+Shift+Delete）
- [ ] **重新登录系统**
- [ ] **更换浏览器**（推荐 Chrome/Edge）
- [ ] **检查网络连接**

---

## 常见错误及解决方法

### 登录已过期

**现象**：提示"登录已过期"或"请重新登录"

**解决方法**：
1. 点击"重新登录"按钮
2. 或刷新页面后重新输入账号密码登录

---

### 无权限访问

**现象**：提示"权限不足"或"无权访问"

**解决方法**：
1. 确认登录的账号是否正确
2. 联系管理员确认您的角色权限
3. 如权限已调整，重新登录后生效

---

### 数据不存在

**现象**：提示"数据不存在"或"记录未找到"

**解决方法**：
1. 检查访问的记录ID是否正确
2. 该记录可能已被删除
3. 返回列表页面重新查找

---

### 数据校验失败

**现象**：提示"请填写必填项"或"数据格式错误"

**解决方法**：
1. 检查页面中的红色提示
2. 确保所有必填项（带红色星号）已填写
3. 检查输入内容的格式是否正确

---

### 网络请求失败

**现象**：提示"网络错误"或"请求超时"

**解决方法**：
1. 检查网络连接状态
2. 刷新页面后重试
3. 如持续失败，联系管理员检查服务器状态

---

## 页面加载缓慢解决方法

### 问题现象
- 页面打开需要很长时间
- 列表数据加载缓慢
- 页面操作卡顿

### 可能原因及解决方法

**原因1：浏览器缓存过多**
- **解决方法**：
  1. 按 **Ctrl+Shift+Delete** 打开清除缓存窗口
  2. 选择"缓存的图像和文件"
  3. 时间范围选择"所有时间"
  4. 点击"清除数据"

**原因2：网络连接不稳定**
- **解决方法**：
  1. 检查网络连接状态
  2. 尝试访问其他网站确认网络正常
  3. 切换网络环境（如WiFi换4G/5G）

**原因3：浏览器插件冲突**
- **解决方法**：
  1. 尝试使用**无痕模式**（Ctrl+Shift+N）打开系统
  2. 如正常，说明有插件冲突
  3. 逐一禁用浏览器插件排查

**原因4：数据量过大**
- **解决方法**：
  1. 列表页面使用**筛选条件**减少显示数据
  2. 分页查看，不要一次加载全部数据

**原因5：浏览器版本过低**
- **解决方法**：
  1. 升级浏览器到最新版本
  2. 推荐使用 **Chrome 90+** 或 **Edge 90+**

## 数据保存失败排查步骤

### 问题现象
- 点击保存按钮无响应
- 提示"保存失败"
- 提示"网络错误"

### 可能原因及解决方法

**原因1：网络中断**
- **解决方法**：
  1. 检查网络连接图标是否正常
  2. 等待网络恢复后**重新保存**
  3. 重要数据建议先**复制到本地备份**

**原因2：必填项未填写**
- **解决方法**：
  1. 检查页面中的**红色星号**必填项
  2. 查看是否有**红色提示文字**
  3. 补充完整后再次保存

**原因3：数据格式错误**
- **解决方法**：
  1. 检查数字字段是否输入了文字
  2. 检查日期格式是否正确
  3. 检查特殊字符是否超出限制

**原因4：会话已过期**
- **解决方法**：
  1. **重新登录**系统
  2. 重新进入编辑页面
  3. 重新填写并保存

## 导出功能异常解决

### 问题现象
- 点击导出无反应
- 导出文件损坏
- 提示"导出失败"

### 可能原因及解决方法

**原因1：浏览器阻止了下载**
- **解决方法**：
  1. 检查浏览器地址栏右侧是否有**下载拦截提示**
  2. 点击提示，选择"始终允许"
  3. 或进入浏览器设置，关闭下载拦截

**原因2：未选择数据**
- **解决方法**：
  1. 导出前需要先勾选需要导出的记录
  2. 勾选记录后导出按钮才会可用

**原因3：浏览器内存不足**
- **解决方法**：
  1. 关闭其他不用的浏览器标签页
  2. 重启浏览器后重试

---

## 打印格式错乱解决

### 问题现象
- 打印预览排版错乱
- 打印内容被截断
- 打印缺少部分内容

### 可能原因及解决方法

**原因1：打印设置不正确**
- **解决方法**：
  1. 打印前使用 **Ctrl+P** 预览
  2. 选择正确的**纸张大小**（推荐A4）
  3. 设置**边距**为"默认"或"最小"
  4. 勾选"背景图形"选项

**原因2：浏览器缩放比例不当**
- **解决方法**：
  1. 按 **Ctrl+0** 恢复默认缩放比例
  2. 打印预览中调整缩放比例至合适大小

**原因3：使用了不兼容的浏览器**
- **解决方法**：
  1. 使用 **Chrome** 或 **Edge** 浏览器打印
  2. 避免使用360浏览器兼容模式

## 浏览器兼容性说明

### 推荐浏览器

| 浏览器 | 推荐版本 | 兼容性评级 |
|-------|---------|-----------|
| Google Chrome | 90+ | 最佳 |
| Microsoft Edge | 90+ | 最佳 |
| Mozilla Firefox | 88+ | 良好 |
| Safari | 14+ | 良好 |

### 不支持的浏览器

| 浏览器 | 说明 |
|-------|------|
| Internet Explorer 11 | 完全不支持，请升级浏览器 |
| Internet Explorer 10及以下 | 完全不支持 |
| 360浏览器兼容模式 | 不支持，请切换至极速模式 |
| 搜狗浏览器兼容模式 | 不支持，请切换至高速模式 |

### 浏览器设置建议

**启用JavaScript**：
系统需要JavaScript支持，请确保浏览器未禁用JS

**启用Cookie**：
系统使用Cookie保存登录状态，请确保Cookie已启用
