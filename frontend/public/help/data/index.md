# 基础数据

基础数据模块用于管理系统运行所需的基础信息，是成本分析的数据基础。

## 功能概述

- 原料管理：维护原料信息和价格
- 包材管理：维护包材配置和价格
- 工序管理：维护生产工序和单价
- 型号管理：维护产品型号信息
- 法规管理：维护法规标准信息
- 客户管理：维护客户信息

## 数据关系

```
型号 → 关联包材配置
     → 关联工序配置

成本分析 → 引用原料
         → 引用包材
         → 引用工序
         → 引用法规
```

## 重要性

基础数据的准确性直接影响成本分析的结果，请确保：
- 价格信息及时更新
- 数据状态正确（启用/停用）
- 关联关系配置正确
