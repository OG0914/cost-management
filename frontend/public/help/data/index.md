# 基础数据

基础数据模块用于管理系统运行所需的基础信息，是成本分析的数据基础。

## 功能概述

- **物料管理**：维护原料和包材的信息和价格
- **工序管理**：维护生产工序和单价
- **型号管理**：维护产品型号信息和BOM配置
- **法规管理**：维护法规标准信息
- **客户管理**：维护客户信息
- **包装管理**：按型号配置包材和工序

## 数据关系

```
型号 → 关联法规
    → 配置BOM（原料清单）
    → 关联包材配置（按型号）
    → 关联工序配置（按型号+包装类型）

成本分析 → 引用型号
         → 引用原料/包材/工序
         → 引用法规
         → 引用客户
```

## 重要性

基础数据的准确性直接影响成本分析的结果，请确保：
- 价格信息及时更新
- 数据状态正确（启用/停用）
- 关联关系配置正确
