# 原料管理

## 功能概述

原料管理用于维护系统中所有原料的基础信息和价格，支持按产品类型（半面罩类/口罩类）分类管理，是成本分析的基础数据来源。

---

## 页面布局

页面分为筛选区、列表区和操作区：
- **筛选区**：搜索框、品名类别筛选、细分类筛选
- **列表区**：原料数据表格（半面罩类和非半面罩类显示不同列）
- **操作区**：右上角工具栏（展开后显示全部功能按钮）

---

## 原料信息字段

### 公共字段（所有类型）

| 字段名 | 说明 |
|--------|------|
| 品名类别 | 原料/包材 |
| 细分类 | 按产品类型划分的子类别 |
| 品号 | 原料的唯一编码 |
| 原料名称 | 原料的标准名称 |
| 单价 | 当前采购单价 |
| 币别 | 货币单位（CNY/USD等） |
| 单位 | 计量单位 |
| 更新时间 | 最后修改时间 |

### 半面罩类特有字段

| 字段名 | 说明 |
|--------|------|
| 供应商 | 原料供应商名称 |
| 绑定型号 | 该原料绑定的产品型号 |
| 包装方式 | 原料的包装方式 |
| 用量 | 单件产品的用量 |
| 生产周期 | 原料生产周期 |
| MOQ | 最小起订量 |

### 非半面罩类（口罩类/通用）特有字段

| 字段名 | 说明 |
|--------|------|
| 厂商 | 生产厂家 |
| 备注 | 补充说明 |

## 操作指南

### 1. 新增原料

1. 点击页面右上角 **【新增原料】** 按钮
2. 填写原料基本信息（根据产品类型显示不同表单字段）
3. 点击 **【保存】** 完成创建

### 2. 编辑原料

1. 在原料列表中找到目标原料
2. 点击操作列的 **【编辑】** 按钮
3. 修改需要更新的字段
4. 点击 **【保存】** 确认修改

### 3. 删除原料

1. 找到需要删除的原料
2. 点击操作列的 **【删除】** 按钮
3. 确认删除操作

### 4. 批量删除

1. 在列表中勾选需要删除的原料（使用左侧复选框）
2. 点击工具栏的 **【批量删除】** 按钮
3. 确认删除操作

### 5. 导出Excel

1. 在列表中勾选需要导出的原料
2. 点击工具栏的 **【导出Excel】** 按钮
3. 系统自动下载选中数据为Excel文件

### 6. 导入Excel

1. 点击工具栏的 **【下载模板】** 获取导入模板（根据当前产品类型下载对应模板）
2. 按模板格式填写原料数据
3. 点击 **【导入Excel】** 按钮，选择填写好的文件
4. 系统预检查，如发现重复品号会弹出确认对话框：
   - **取消导入**：放弃本次导入
   - **跳过重复项**：保留系统现有数据，跳过重复项
   - **覆盖更新**：用新数据更新重复品号
5. 导入完成后显示结果（新增/更新/跳过条数）

## 筛选功能

### 搜索框
输入品号、名称或厂商进行模糊搜索（防抖300ms自动触发）。

### 品名类别筛选
- **原料**：只显示原料类数据
- **包材**：只显示包材类数据

### 细分类筛选（可拖拽排序）
根据当前产品类型显示对应的细分类标签：
- 点击标签可筛选特定细分类的原料
- 标签支持拖拽排序，自定义显示顺序（排序结果保存在本地存储）
- "全部"标签固定在最前面，不参与排序

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 品号唯一性 | 品号不能重复，导入时会检测重复 |
| 导入模板类型 | 根据当前产品类型下载对应模板（半面罩类/口罩类/通用） |
| 细分类排序 | 拖拽排序结果保存在浏览器本地存储中 |
| 删除影响 | 已被成本分析引用的原料不建议删除 |
| 重复处理 | 导入支持跳过或覆盖更新重复品号 |
