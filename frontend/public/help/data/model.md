# 型号管理

## 功能概述

型号管理用于维护产品型号的基础信息，以及型号与BOM（物料清单）的关联配置，是成本分析的核心基础数据。

---

## 页面布局

页面分为筛选区、数据展示区和操作区：
- **筛选区**：搜索框（支持型号名称、法规类别搜索）、视图切换按钮
- **数据展示区**：卡片视图或列表视图展示型号数据
- **操作区**：右上角工具栏（展开后显示全部功能按钮）

---

## 型号信息字段

| 字段名 | 说明 |
|--------|------|
| 法规类别 | 该型号适用的法规标准 |
| 型号名称 | 产品的标准名称 |
| 产品系列 | 产品所属系列（如MK81） |
| 型号分类 | 口罩/半面罩/全面罩/PPE产品 |
| BOM数量 | 该型号配置的BOM物料项数 |
| 状态 | 已启用/已禁用 |
| 产品图片 | 型号的产品展示图片 |

## 操作指南

### 1. 新增型号

1. 点击页面右上角 **【新增型号】** 按钮
2. 填写型号基本信息：
   - **法规类别**（必填）：选择适用的法规标准
   - **型号名称**（必填）：产品的标准名称
   - **产品系列**（必填）：输入产品系列（支持从现有系列中选择）
   - **型号分类**（必填）：选择产品分类（口罩/半面罩/全面罩/PPE产品）
3. 点击 **【确定】** 完成创建

### 2. 编辑型号

1. 在型号列表中找到目标型号
2. 点击操作列的 **【编辑】** 按钮
3. 修改需要更新的字段：
   - 基础信息：法规类别、型号名称、产品系列、型号分类
   - 状态：启用/禁用
   - 产品图片：上传/管理产品图片
4. 点击 **【确定】** 确认修改

### 3. 产品图片管理

在编辑型号弹窗中管理图片：

1. **上传图片**：
   - 点击上传按钮选择图片
   - 支持格式：JPG、PNG、WEBP
   - 单张最大5MB
   - 支持上传多张图片

2. **设置主图**：
   - 点击 **【设为主图】** 按钮将某张图片设为主图
   - 主图会显示在卡片视图和列表视图中

3. **删除图片**：
   - 点击 **【删除】** 按钮删除不需要的图片

### 4. 配置BOM

1. 在型号卡片或列表中找到目标型号
2. 点击 **【配置BOM】** 按钮（或点击BOM数量链接）
3. 在BOM配置弹窗中：
   - 添加物料：选择物料并设置用量
   - 编辑物料：修改用量或信息
   - 删除物料：移除不需要的物料
4. 点击 **【保存】** 完成配置

### 5. 删除型号

1. 找到需要删除的型号
2. 点击操作列的 **【删除】** 按钮
3. 确认删除操作

### 6. 批量删除

1. 在列表视图中勾选需要删除的型号（使用左侧复选框）
2. 点击工具栏的 **【批量删除】** 按钮
3. 确认删除操作

### 7. 导出Excel

1. 在列表视图中勾选需要导出的型号
2. 点击工具栏的 **【导出Excel】** 按钮
3. 系统自动下载选中数据为Excel文件

### 8. 导入Excel

1. 点击工具栏的 **【下载模板】** 获取导入模板
2. 按模板格式填写型号数据
3. 点击 **【导入Excel】** 按钮，选择填写好的文件
4. 系统校验后完成导入，显示导入结果（创建条数、更新条数、错误信息）

## 视图模式

### 卡片视图
- 以卡片形式展示型号
- 显示产品图片（或占位图标）、型号名称、法规类别、产品系列、型号分类、BOM数量、状态
- 操作按钮：配置BOM、编辑、删除

### 列表视图
- 以表格形式展示型号
- 支持多选（用于批量删除和导出）
- 显示更多信息：产品图、法规类别、型号名称、产品系列、型号分类、BOM明细、状态

## 搜索与筛选

### 搜索框
支持按型号名称、法规类别、产品系列进行模糊搜索（防抖300ms自动触发）。

## 型号分类说明

| 分类 | 说明 |
|------|------|
| 口罩 | N95/KN95等防护口罩 |
| 半面罩 | 半面罩呼吸防护器 |
| 全面罩 | 全面罩呼吸防护器 |
| PPE产品 | 个人防护设备 |

---

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 法规类别 | 创建型号前需先在【法规管理】中配置法规类别 |
| 产品系列 | 支持从历史数据中快速选择，也可输入新系列 |
| BOM配置 | 型号创建后需配置BOM才能在成本分析中使用 |
| 状态影响 | 禁用的型号在新建成本分析时无法选择 |
| 图片格式 | 只支持JPG、PNG、WEBP格式，单张最大5MB |
