# 包材管理

## 功能概述

包材管理用于按型号配置包装材料和包装方式，支持多层级包装结构设置，实现成本分析时的自动计算。

---

## 页面布局

页面分为筛选区、数据展示区和操作区：
- **筛选区**：产品类别、型号、包装类型筛选，视图切换按钮
- **数据展示区**：卡片视图或列表视图展示包装配置
- **操作区**：右上角工具栏（展开后显示全部功能按钮）

---

## 包装配置信息字段

| 字段名 | 说明 |
|--------|------|
| 产品类别 | 型号所属的产品分类 |
| 型号 | 关联的产品型号 |
| 配置名称 | 包装配置的名称标识 |
| 生产工厂 | 生产工厂（东莞迅安/湖北知腾） |
| 包装类型 | 标准装箱/无装箱/吸塑直接装箱/吸塑+袋装 |
| 包装方式 | 各层级的包装数量结构 |
| 每箱数量 | 每个外箱的总数量（PCS） |
| 包材数量 | 该配置包含的包材项数 |
| 工序总价 | 关联工序的总价格 |
| 状态 | 已启用/已禁用 |

---

## 包装类型说明

| 类型 | 说明 | 层级结构 |
|------|------|----------|
| 标准装箱 | 常规袋装+箱装+外箱三层结构 | 袋 → 盒 → 箱 |
| 无装箱 | 不需要外包装 | 单层 |
| 吸塑直接装箱 | 吸塑后直接装入外箱 | 吸塑 → 箱 |
| 吸塑+袋装 | 吸塑后袋装再装箱 | 吸塑 → 袋 → 箱 |

## 操作指南

### 1. 查看包材明细

1. 在包装配置卡片或列表中找到目标配置
2. 点击 **【查看】** 按钮（或包材数量链接）
3. 在弹窗中查看该配置的所有包材明细：
   - 包材名称
   - 基本用量
   - 单价
   - 外箱体积

### 2. 新增包装配置

1. 点击页面右上角工具栏区域的新增按钮
2. 填写包装配置信息：
   - **型号**（必填）：选择关联的产品型号
   - **配置名称**（必填）：包装配置的名称
   - **生产工厂**：选择生产工厂（东莞迅安/湖北知腾）
   - **包装类型**：选择包装方式类型
   - **包装方式**：填写各层级数量
     - 第一层：每袋/每吸塑数量
     - 第二层：每盒/每袋数量
     - 第三层：每箱数量（如适用）
   - **状态**：启用/禁用
3. 添加包材：
   - 点击 **【添加包材】** 按钮
   - 输入包材名称（支持从原料库搜索选择）
   - 系统自动填充单价
   - 填写基本用量和外箱体积
4. 点击 **【确定】** 完成创建

### 3. 编辑包装配置

1. 在包装配置列表中找到目标配置
2. 点击操作列的 **【编辑】** 按钮
3. 修改需要更新的字段：
   - 基础信息：型号、配置名称、生产工厂、包装类型
   - 包装方式：各层级数量
   - 包材：添加、删除、修改包材信息
   - 状态：启用/禁用
4. 点击 **【确定】** 确认修改

### 4. 一键复制包材

1. 在编辑包装配置弹窗中
2. 点击 **【一键复制包材】** 按钮
3. 选择复制源：从其他有包材的配置中选择
4. 选择复制模式：
   - **替换**：清空现有包材，使用源配置的全部包材
   - **合并**：追加源配置的包材，跳过重复的包材
5. 点击 **【确定】** 完成复制

### 5. 删除包装配置

1. 找到需要删除的包装配置
2. 点击操作列的 **【删除】** 按钮
3. 确认删除操作

### 6. 批量删除

1. 在列表视图中勾选需要删除的配置（使用左侧复选框）
2. 点击工具栏的 **【批量删除】** 按钮
3. 确认删除操作
4. 如部分配置被引用无法删除，会显示删除结果

### 7. 导出Excel

1. 在列表视图中勾选需要导出的配置
2. 点击工具栏的 **【导出Excel】** 按钮
3. 系统自动下载选中数据为Excel文件

### 8. 导入Excel

1. 点击工具栏的 **【下载模板】** 获取导入模板
2. 按模板格式填写包材数据
3. 点击 **【导入Excel】** 按钮，选择填写好的文件
4. 系统校验后完成导入，显示导入结果（创建条数、更新条数、错误信息）

## 筛选功能

### 产品类别筛选
选择产品类别后，型号下拉框只显示该类别下的型号。

### 型号筛选
选择具体型号，只显示该型号的包装配置。

### 包装类型筛选
按包装类型筛选配置（标准装箱/无装箱/吸塑直接装箱/吸塑+袋装）。

---

## 视图模式

### 卡片视图
- 以卡片形式展示包装配置
- 显示型号、配置名称、生产工厂、包装类型、包装方式、每箱数量、包材数量、状态
- 操作按钮：查看、编辑、删除
- 只显示启用的配置

### 列表视图
- 以表格形式展示包装配置
- 支持多选（用于批量删除和导出）
- 显示更多信息：产品类别、型号、配置名称、生产工厂、包装类型、包装方式、每箱数量、工序总价、状态、创建时间
- 显示全部配置（包括禁用的）

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 型号关联 | 包装配置必须关联一个型号 |
| 包材来源 | 包材从原料管理中 category=包材 的数据中选择 |
| 层级验证 | 不同包装类型需要填写对应层级数量 |
| 删除限制 | 已被成本分析引用的配置无法删除 |
| 复制包材 | 一键复制功能可以快速复用相似配置的包材 |
