# 工序管理

## 功能概述

工序管理用于按型号和包装类型配置生产工序流程，支持为不同产品配置不同的工序组合，是成本分析中人工成本计算的基础数据来源。

---

## 页面布局

页面分为筛选区、数据展示区和操作区：
- **筛选区**：产品类别、型号、包装类型筛选，视图切换按钮
- **数据展示区**：卡片视图或列表视图展示工序配置
- **操作区**：右上角工具栏（展开后显示全部功能按钮）

---

## 工序配置信息字段

| 字段名 | 说明 |
|--------|------|
| 产品类别 | 型号所属的产品分类 |
| 型号 | 关联的产品型号 |
| 配置名称 | 工序配置的名称标识 |
| 生产工厂 | 生产工厂（东莞迅安/湖北知腾） |
| 包装类型 | 关联的包装配置类型 |
| 包装方式 | 该配置使用的包装结构 |
| 每箱数量 | 每个外箱的总数量（PCS） |
| 工序总价 | 该配置所有工序的价格合计 |
| 状态 | 已启用/已禁用 |
| 创建时间 | 配置创建时间 |

## 操作指南

### 1. 查看工序明细

1. 在工序配置卡片或列表中找到目标配置
2. 点击 **【查看】** 按钮
3. 在弹窗中查看该配置的工序明细：
   - 工序名称
   - 工序单价
   - 工序描述

### 2. 新增工序配置

1. 点击页面右上角 **【新增工序配置】** 按钮
2. 在工序配置弹窗中填写信息：
   - **型号**（必填）：选择关联的产品型号
   - **配置名称**（必填）：工序配置的名称
   - **生产工厂**：选择生产工厂（东莞迅安/湖北知腾）
   - **包装类型**：选择关联的包装类型
   - **包装方式**：配置包装层级数量
   - **状态**：启用/禁用
3. 添加工序：
   - 在工序表格中添加行
   - 选择工序名称
   - 设置工序单价
4. 点击 **【保存】** 完成创建

### 3. 编辑工序配置

1. 在工序配置列表中找到目标配置
2. 点击操作列的 **【编辑】** 按钮
3. 修改需要更新的字段：
   - 基础信息：型号、配置名称、生产工厂、包装类型
   - 包装方式：各层级数量
   - 工序：添加、删除、修改工序信息
   - 状态：启用/禁用
4. 点击 **【保存】** 确认修改

### 4. 删除工序配置

1. 找到需要删除的工序配置
2. 点击操作列的 **【删除】** 按钮
3. 确认删除操作

### 5. 批量删除

1. 在列表视图中勾选需要删除的配置（使用左侧复选框）
2. 点击工具栏的 **【批量删除】** 按钮
3. 确认删除操作
4. 如部分配置无法删除，会显示具体原因

### 6. 导出Excel

1. 在列表视图中勾选需要导出的配置
2. 点击工具栏的 **【导出Excel】** 按钮
3. 系统自动下载选中数据为Excel文件

### 7. 导入Excel

1. 点击工具栏的 **【下载模板】** 获取导入模板
2. 按模板格式填写工序数据
3. 点击 **【导入Excel】** 按钮，选择填写好的文件
4. 系统校验后完成导入，显示导入结果（创建条数、更新条数、错误信息）

## 筛选功能

### 产品类别筛选
选择产品类别后，型号下拉框只显示该类别下的型号。

### 型号筛选
选择具体型号，只显示该型号的工序配置。

### 包装类型筛选
按包装类型筛选配置（标准装箱/无装箱/吸塑直接装箱/吸塑+袋装）。

---

## 视图模式

### 卡片视图
- 以卡片形式展示工序配置
- 显示型号、配置名称、生产工厂、包装类型、包装方式、工序总价、状态
- 操作按钮：查看、编辑、删除
- 只显示启用的配置

### 列表视图
- 以表格形式展示工序配置
- 支持多选（用于批量删除和导出）
- 显示更多信息：产品类别、型号、配置名称、生产工厂、包装类型、包装方式、每箱数量、工序总价、状态、创建时间
- 显示全部配置（包括禁用的）

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 型号关联 | 工序配置必须关联一个型号和包装类型 |
| 与包装关系 | 工序配置与包装配置通过型号和包装类型关联 |
| 删除限制 | 已被成本分析引用的配置无法删除 |
| 工序单价 | 工序单价用于成本分析时计算人工成本 |
| 包装方式 | 包装方式数据来自关联的包装配置 |
