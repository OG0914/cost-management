# 客户管理

## 功能概述

客户管理用于维护客户的基础信息，包括VC号、客户名称、地区和负责业务，用于成本分析中的客户选择。

---

## 页面布局

页面分为筛选区、列表区和操作区：
- **筛选区**：搜索框（支持VC号、客户名称、地区模糊搜索）
- **列表区**：客户数据表格
- **操作区**：右上角工具栏（展开后显示全部功能按钮）

---

## 客户信息字段

| 字段名 | 说明 |
|--------|------|
| VC号 | 客户的唯一标识编码 |
| 客户名称 | 客户的完整名称 |
| 地区 | 客户所在地区 |
| 负责业务 | 负责该客户的业务员（未分配显示"公共"） |
| 备注 | 补充说明 |
| 更新时间 | 最后修改时间 |

---

## 操作指南

### 1. 新增客户

1. 点击页面右上角 **【新增客户】** 按钮
2. 填写客户信息：
   - **VC号**（必填）：客户的唯一标识
   - **客户名称**（必填）：客户的完整名称
   - **地区**：客户所在地区
   - **负责业务**：选择负责该客户的业务员（管理员可分配，不选则为公共客户）
   - **备注**：补充说明
3. 点击 **【确定】** 完成创建

### 2. 编辑客户

1. 在客户列表中找到目标客户
2. 点击操作列的 **【编辑】** 按钮
3. 修改需要更新的字段
4. 点击 **【确定】** 确认修改

### 3. 删除客户

1. 找到需要删除的客户
2. 点击操作列的 **【删除】** 按钮
3. 确认删除操作

### 4. 批量删除

1. 在列表中勾选需要删除的客户（使用左侧复选框）
2. 点击工具栏的 **【批量删除】** 按钮
3. 确认删除操作

### 5. 导出Excel

1. 在列表中勾选需要导出的客户
2. 点击工具栏的 **【导出Excel】** 按钮
3. 系统自动下载选中数据为Excel文件

⚠️ **注意**：需先勾选数据才能导出，未选择时导出按钮会提示。

### 6. 导入Excel

1. 点击工具栏的 **【下载模板】** 获取导入模板
2. 按模板格式填写客户数据
3. 点击 **【导入Excel】** 按钮，选择填写好的文件
4. 系统校验后完成导入，显示导入结果（创建条数、更新条数）

## 搜索与筛选

### 搜索框
支持按VC号、客户名称、地区进行模糊搜索，输入后自动触发搜索（防抖300ms）。

---

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| VC号唯一性 | VC号不能重复，系统会校验唯一性 |
| 负责业务分配 | 仅管理员可以分配负责业务员，不分配则为公共客户 |
| 删除影响 | 删除客户不会影响已创建的成本分析记录 |
| 导入更新 | 导入时如VC号已存在，会更新现有客户数据 |
