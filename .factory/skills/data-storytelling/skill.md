## Data Storytelling

Transform raw data into compelling narratives that drive action.

### When to Use This Skill

- Creating executive reports
- Building data presentations
- Designing analytics dashboards
- Communicating KPI results
- Presenting performance reviews

### Story Structure

#### 1. The Hook (Context)
- What's the current situation?
- Why does this matter now?
- What question are we answering?

#### 2. The Journey (Analysis)
- What does the data show?
- What patterns emerge?
- What comparisons are relevant?

#### 3. The Insight (Conclusion)
- What's the key takeaway?
- What action should follow?
- What's the impact?

### Storytelling Best Practices

#### Do's
- Start with the insight, not the data
- Use comparisons for context
- Highlight anomalies and patterns
- Include clear call-to-action
- Tailor depth to audience

#### Don'ts
- Don't drown in data details
- Don't use jargon without explanation
- Don't present without recommendations
- Don't ignore negative findings
- Don't forget the "so what?"
