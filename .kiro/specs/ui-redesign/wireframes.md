## 7. 配色方案

```
主色调 (Primary Blue):
├─ primary-50:  #EFF6FF  (浅蓝背景)
├─ primary-100: #DBEAFE  (选中背景)
├─ primary-500: #3B82F6  (标准蓝)
├─ primary-600: #2563EB  (主要按钮)
└─ primary-700: #1D4ED8  (深蓝)

灰色系 (Slate):
├─ slate-50:  #F8FAFC  (页面背景)
├─ slate-100: #F1F5F9  (卡片边框)
├─ slate-200: #E2E8F0  (分割线)
├─ slate-400: #94A3B8  (次要文字/图标)
├─ slate-500: #64748B  (正文)
├─ slate-700: #334155  (标题)
└─ slate-800: #1E293B  (强调文字)

功能色:
├─ green-500:  #22C55E  (成功/正常)
├─ orange-500: #F97316  (警告/待处理)
├─ red-500:    #EF4444  (错误/退出)
└─ purple-500: #A855F7  (信息/统计)
```

## 8. 字体规范

```
字体家族: 'Inter', sans-serif

字号层级:
├─ 2xl (24px): 页面大标题、问候语
├─ xl  (20px): 卡片标题
├─ lg  (18px): 区块标题
├─ base(16px): 正文
├─ sm  (14px): 菜单项、按钮文字
└─ xs  (12px): 辅助信息、标签

字重:
├─ font-bold   (700): 数值、强调
├─ font-semibold(600): 标题
├─ font-medium (500): 菜单项
└─ font-normal (400): 正文
```

---

## 9. 响应式断点 (参考)

```
当前设计为桌面端优先，最小宽度 1024px

未来可扩展:
├─ lg (1024px+): 完整侧边栏
├─ md (768px-1023px): 收起侧边栏，仅显示图标
└─ sm (<768px): 隐藏侧边栏，顶部汉堡菜单
```

## 10. 页面切换动画

```
淡入动画 (fadeIn):
@keyframes fadeIn {
    from { 
        opacity: 0; 
        transform: translateY(10px); 
    }
    to { 
        opacity: 1; 
        transform: translateY(0); 
    }
}
duration: 0.4s
timing: ease-out

子菜单展开动画:
├─ max-height: 0 → 200px
├─ opacity: 0 → 1
├─ duration: 0.3s
└─ timing: ease-in-out
```
