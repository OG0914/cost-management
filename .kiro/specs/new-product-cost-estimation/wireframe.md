# 新产品成本预估 - 界面草图

## 1. 导航菜单结构 - 方案A（二级菜单）

```
┌─────────────────────────────────────┐
│  侧边栏菜单                          │
├─────────────────────────────────────┤
│  📊 仪表盘                           │
│  📋 成本管理                         │
│    ├─ 📝 新增报价单 ▼               │
│    │    ├─ 现有产品报价  ← 跳转到标准成本页面，让用户选择后复制新增
│    │    └─ 新产品成本预估 ← 跳转到新产品预估页面
│    ├─ 📄 报价单列表                 │
│    └─ 💰 标准成本                   │
│  ...                                │
└─────────────────────────────────────┘
```

**点击"现有产品报价"：**
- 跳转到标准成本页面
- 业务员选择型号的标准成本
- 点击"复制新增"
- 跳转到 CostAdd 页面，数据自动带出

**点击"新产品成本预估"：**
- 跳转到新产品预估页面（新页面，或复用 CostAdd 加参数）
- 业务员选择新产品型号
- 选择参考标准成本
- 数据自动带出
- 填写客户信息，调整数据，保存

## 4. 标记说明

| 标记 | 含义 |
|------|------|
| 🏷️参考 | 数据来自参考标准成本，未修改 |
| 🏷️参考 ✏️已改 | 数据来自参考标准成本，已被用户修改 |
| (无标记) | 用户新增的数据行 |
