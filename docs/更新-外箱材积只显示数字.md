# 更新说明 - 外箱材积只显示数字

## 更新日期
2025-11-09

## 更新内容

外箱材积列只显示数字，不显示单位"立方英尺"。

## 修改对比

### 修改前
```
外箱材积：1.5 立方英尺
```

### 修改后
```
外箱材积：1.5
```

## 修改文件

### 1. 报价单新增/编辑页面
**文件：** `frontend/src/views/cost/CostAdd.vue`

**修改前：**
```vue
<span v-if="row.carton_volume">{{ row.carton_volume }} 立方英尺</span>
```

**修改后：**
```vue
<span v-if="row.carton_volume">{{ row.carton_volume }}</span>
```

### 2. 报价单详情页面
**文件：** `frontend/src/views/cost/CostDetail.vue`

**修改前：**
```vue
<span v-if="row.carton_volume">{{ row.carton_volume }} 立方英尺</span>
```

**修改后：**
```vue
<span v-if="row.carton_volume">{{ row.carton_volume }}</span>
```
