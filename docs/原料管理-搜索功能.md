# 原料管理 - 搜索功能说明

## 功能概述

在原料管理页面新增了关键词搜索功能，方便用户快速查找原料。

## 功能特性

### 关键词搜索
- **搜索范围**: 品号、原料名称
- **搜索方式**: 实时搜索，输入即过滤
- **大小写**: 不区分大小写
- **清除**: 点击清除按钮可清空搜索条件

## 界面位置

搜索栏位于原料管理页面的表格上方，包含一个搜索输入框（带搜索图标）。

## 使用方法

### 按关键词搜索
1. 在搜索框中输入品号或原料名称的关键词
2. 表格自动过滤显示匹配的原料
3. 点击清除按钮可清空搜索

## 技术实现

### 前端实现
- **文件**: `frontend/src/views/material/MaterialManage.vue`
- **搜索逻辑**: 客户端过滤，实时响应
- **数据源**: 从 `materials` 过滤到 `filteredMaterials`

### 搜索算法
```javascript
// 关键词搜索（品号或原料名称）
if (searchKeyword.value) {
  const keyword = searchKeyword.value.toLowerCase()
  result = result.filter(item => 
    item.item_no.toLowerCase().includes(keyword) || 
    item.name.toLowerCase().includes(keyword)
  )
}
```

## 修改文件

## 注意事项

1. 搜索是客户端过滤，不会重新请求后端数据
2. 搜索结果实时更新，无需点击搜索按钮
3. 清空搜索条件后会显示所有原料
4. 搜索功能简洁高效，专注于快速查找
